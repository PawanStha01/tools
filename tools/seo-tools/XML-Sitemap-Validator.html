Quick Tips
About XML Sitemaps

XML sitemaps help search engines discover and index your website's pages.

Sitemap Requirements
  • Must be UTF-8 encoded
  • Must use correct XML syntax
  • URLs must be properly escaped
  • Should be less than 50MB (uncompressed)
  • Should contain less than 50,000 URLs
Common Issues
  • Missing XML declaration
  • Invalid characters in URLs
  • Incorrect namespace declarations
  • Missing required tags (like <loc>)
  • Invalid date formats
